# macOS Tips

## Shortcut

* `⌘` + `Tab`: Switch application
* `⌘` + <code>`</code>: Cycle between open windows in the selected application
* `⌘` + `H`: Hide the window
* `⌘` + `M`: Minimize the window
* `⌘` + `N`: Create a new window
* `⌘` + `O`: Open files
* `⌘` + `S`: Save
* `⌘` + `⇧` + `S`: Save as
* `⌘` + `P`: Print the window
* `⌘` + `W`: Close
* `⌘` + `P`: Quit
* `⌘` + `⌫`: Move to trash
